## Getting Started

First, install all dependencies:

```bash
npm install
```

Create an .env based on .env.example:

```
NEXT_PUBLIC_GRAPHQL_URL=""
```

Now, you can run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Or, build a production version with:

```bash
npm run build
```

Then, execute it with:

```bash
npm run start
```
